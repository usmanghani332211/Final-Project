# Maritime Accident Injury Prediction 🚢

## Overview
This project uses machine learning to predict injury types in UK maritime accidents using MAIB data (2010–2024).

## Models Used
- Logistic Regression  
- Random Forest  
- Gradient Boosting  
- MLP Neural Network  
- Deep Neural Network  

## Methods
- Data cleaning and preprocessing  
- One-hot encoding and feature scaling  
- Class imbalance handling (merging + undersampling)  
- 60/40 train-test split  
- Hyperparameter tuning (GridSearchCV)  

## Results
- Random Forest performed best (~41% accuracy)  
- Models worked well on common injuries but struggled with rare classes  

## Tech Stack
Python, Pandas, Scikit-learn, TensorFlow, Matplotlib
